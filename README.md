# STSA

**Structured Tone-Space Augmentation for Extremely Low-Resource Machine Translation of Tone-Marked Languages**

## Code Availability

The core implementation will be made publicly available after the paper is accepted.

## Data Sources

1. Hmong data are collected from [online Bible translations](https://www.bible.com/bible/12/JHN.1.ASV?parallel=1525).
2. Iu Mien data are collected from [online Bible translations](https://www.bible.com/bible/12/JHN.1.MWW2022?parallel=233).
3. The Kam dataset was constructed from printed sources with the following ISBNs: `9787811267983`, `9787811268669`, `9787523202722`, and `9787105159031`.
4. Zhuang data are initialized from [ZhuangBench](https://github.com/luciusssss/ZhuangBench).
